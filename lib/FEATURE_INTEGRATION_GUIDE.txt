// XP PROGRESSION:
// ─ Base: 10 XP per game
// ─ Win bonus: +20 XP
// ─ High multiplier bonus: +(multiplier × 5) XP
// ─ Achievement bonus: varies (75-500 XP per achievement)
// ─ Example: Win at 50x = 10 + 20 + 250 = 280 XP

// LEVEL PROGRESSION:
// ─ Level 1 requires: 100 XP
// ─ Level 2 requires: 200 XP
// ─ Level 3 requires: 300 XP
// ─ Level N requires: 100 × N XP
// ─ Max bet unlocked = $100 × Level (Level 5 = $500 max)

// ACHIEVEMENTS (10 total):
// ─ Fifty Times!              (50x cashout) → 100 XP
// ─ Century Club              (100x cashout) → 250 XP
// ─ Legend                    (500x cashout) → 500 XP
// ─ Hot Streak                (5 wins in a row) → 150 XP
// ─ Unstoppable               (10 wins in a row) → 300 XP
// ─ Auto Pilot                (10 auto cashouts) → 75 XP
// ─ Set It and Forget It      (50 auto cashouts) → 200 XP
// ─ Centennial                (100 games played) → 120 XP
// ─ Veteran                   (500 games played) → 350 XP
